Composite Burn Index (CBI) data and field photos collected for the FIRESEV project, western United States

Research Data Product File Index

File Folder Description
_metadata_RDS-2013-0017.html   Metadata file in HTML format containing a description of the content, quality, and other characteristics of the data.
_metadata_RDS-2013-0017.xml   Metadata file in Extensible Markup Language (XML) format containing a description of the content, quality, and other characteristics of the data.
_codes_LocalSpecies_present.csv \Data Comma-delimited ASCII text file containing a list of the species codes found in these data along with the corresponding scientific name.
_Landfire_ESP-PVG_codes.csv \Data Comma-delimited ASCII text file containing a list of potential vegetation group (PVG) codes along with the corresponding description. The codes and descriptions were obtained from http://www.landfire.gov.
_Landfire_EVT_codes.csv \Data Comma-delimited ASCII text file containing a list of existing vegetation type (EVT) codes along with the corresponding description. The codes and descriptions were obtained from http://www.landfire.gov.
firesev_cbi_plots.shp \Data ESRI ArcGIS shapefile and associated files containing locations of the CBI plots for the FIRESEV project, along with fire dates.
FIRESEV_Dataset_All_variables.csv \Data Comma-delimited ASCII text file containing location information, fire information, plot information, vegetation descriptions, biophysical characteristics of each site, name of photos related by plot number, ground cover estimations, percentage estimates of change due to fire, calculated CBI from the percentage estimates, and GeoCBI calculations.
FIRESEV_Dataset_All_variables.xlsx \Data Microsoft Excel spreadsheet containing all information in CSV file, but with working hyperlinks to photos.
FIRESEV_FieldData_Codes.accdb \Data Microsoft Access database containing all of the information in the CSV and XLSX files, along with datacodes and value ranges for most field variables.
PhotoName.jpg \Data\Photos\FireName JPEG image files (37 folders containing a total of 1090 images) containing pictures of ground cover view of the stand (labeled Ground) or the overall stand structure (labeled Stand). Most of the photos are taken looking north (labeled N). (FireName = name of fire and PhotoName = plotID_N_Ground or plotID_N_Stand typically, but there are occasionally files without the "N" or filenames contains "Extra" which means the photos are extra pictures)
cbi_recode_110211.r \Supplements "R" software script file containing the script used to convert FIRESEV estimates of percent fuel consumed for each criteria and stratum to standard Key and Benson (2006) CBI format.
FIRESEV_all_cbi_plots_graphic.jpg \Supplements JPEG image file containing a map of CBI plots coded as those collected for the FIRESEV project.
FIRESEV_field_forms.pdf \Supplements Adobe Acrobat PDF/a file containing the FIRESEV Burn Severity and Plot Description field forms used to collect the data available in this data product.
FIRESEV_field_methods_description.pdf \Supplements Adobe Acrobat PDF file containing a complete description of the methods and protocols used to collect FIRESEV data for CBI evaluations. (This document also includes the blank field forms for Burn Severity and Plot Description.)
